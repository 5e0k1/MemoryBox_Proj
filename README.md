# 📦 MemoryBox

> 소규모 그룹을 위한 이미지 · 영상 공유 모바일 웹 플랫폼
> **AWS 기반 인프라 구축 및 운영 경험을 위한 개인 프로젝트**

---

## 📌 프로젝트 개요

MemoryBox는 지인들과 흩어져 있는 사진과 영상을 한 곳에 모아
**쉽게 업로드 · 조회 · 다운로드할 수 있는 피드형 웹 서비스**입니다.

동시에, 단순한 웹 개발이 아닌
👉 **AWS 인프라 구축 / 배포 / 운영 경험 확보**를 주요 목표로 합니다.

---

## 🎯 프로젝트 목표

### 1. 실사용 서비스 구현

* 약 7명의 지인 그룹 대상
* 이미지 및 영상 공유/관리
* 모바일 환경 중심 UX 제공

### 2. 인프라 실무 경험 확보

* AWS 기반 서버 구축 및 배포
* S3 스토리지 설계
* CDN 적용 (CloudFront)
* 트래픽 및 비용 최적화 경험

---

## 👥 대상 사용자

* 소규모 그룹 (약 7명)
* 모바일 사용 중심
* IT 비숙련 사용자 → 직관적인 UI 필요

## 🚀 주요 기능 (MVP)

### 📁 파일 관리

* 이미지 / 영상 업로드
* 원본 다운로드
* 파일 삭제

### 🖼 피드형 조회

* 인스타그램 스타일 피드 UI
* 썸네일 기반 리스트
* 상세보기 화면 제공

### 🏷 태그 및 필터

* 태그 추가
* 태그 기반 필터링
* 날짜 / 업로드 기준 정렬

---

## 🗂 파일 저장 구조

| 구분       | 용도   | 포맷    |
| -------- | ---- | ----- |
| Small    | 썸네일  | WebP  |
| Medium   | 상세보기 | WebP  |
| Original | 원본   | 원본 유지 |

### 🎯 목적

* 트래픽 비용 절감
* 빠른 로딩 속도 확보
* 사용자 경험 개선

## 🏗 인프라 구성

### 기본 구조

* Backend: Spring Boot
* Storage: AWS S3
* Deployment: AWS EC2 (Docker 기반)
* CDN: AWS CloudFront (예정)
